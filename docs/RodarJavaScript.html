Guia para Rodar JavaScript no Visual Studio Code (VSCode)

1. Instalar Node.js

Primeiro, você precisa ter o Node.js instalado no seu computador. O Node.js permite executar código JavaScript fora do navegador. Você pode baixar e instalar o Node.js a partir do site oficial.

2. Verificar Instalação do Node.js

Após a instalação, você pode verificar se o Node.js está corretamente instalado abrindo o terminal e digitando:

node -v

Isso deve retornar a versão do Node.js instalada.

3. Instalar o VSCode

Caso ainda não tenha o Visual Studio Code instalado, você pode baixá-lo e instalá-lo a partir do site oficial.

4. Configurar o Ambiente no VSCode

  1. Abra o VSCode.
  2. Crie uma nova pasta para seu projeto ou abra uma pasta existente onde você deseja salvar seus arquivos JavaScript.
  3. Crie um novo arquivo JavaScript. Por exemplo, app.js.

5. Escrever o Código JavaScript

No seu arquivo app.js, você pode escrever o código JavaScript que deseja executar. Por exemplo:

console.log('Hello, world!');

6. Executar o Código JavaScript

  1. Abra o terminal integrado no VSCode. Você pode fazer isso clicando em Terminal > New Terminal no menu superior ou usando o atalho de teclado (Ctrl + `).
  2. No terminal, navegue até o diretório onde seu arquivo app.js está salvo (se necessário).
  3. Digite o comando para executar o arquivo JavaScript com o Node.js:
node app.js

Isso deve exibir a saída Hello, world! no terminal.

7. Extensões Úteis

Existem algumas extensões que podem melhorar sua experiência ao trabalhar com JavaScript no VSCode:

  • Node.js Extensions: Traz um pacote para usar com o node.
  • ESLint: Para linting de código JavaScript.
  • Prettier - Code formatter: Para formatação de código.
  • Debugger for Chrome: Para depuração de código JavaScript no Chrome.

8. Depuração de Código

Você também pode depurar seu código JavaScript diretamente no VSCode:

  1. Configure o launch.json para Node.js. Vá em Run > Add Configuration... e selecione Node.js.
  2. Coloque pontos de interrupção (breakpoints) clicando na margem esquerda ao lado do número da linha no seu arquivo de código.
  3. Inicie a depuração clicando no ícone de "play" na seção de depuração ou usando o atalho (F5).

Seguindo esses passos, você será capaz de rodar e depurar código JavaScript no Visual Studio Code.

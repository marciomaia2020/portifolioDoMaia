Plataformas de Desenvolvimento Sem Codificação


Kodular

Kodular é uma plataforma para criar aplicativos Android sem programação.

Appian

Appian é uma plataforma de automação de processos empresariais com desenvolvimento de aplicativos de baixo código.

Appy Pie

Appy Pie permite criar aplicativos móveis para Android e iOS com uma interface intuitiva de arrastar e soltar.

Thunkable

Thunkable é uma ferramenta para criar aplicativos nativos para Android e iOS usando um editor visual.

Adalo

Adalo é uma plataforma para criar aplicativos móveis e web apps sem codificação.

Bubble

Bubble é ideal para desenvolvimento de aplicativos web com um editor visual baseado em arrastar e soltar.

Glide

Glide permite criar aplicativos móveis com planilhas do Google Sheets como backend, sem codificação.

BuildFire

BuildFire é uma plataforma para construção de aplicativos móveis personalizados sem programação.

AppSheet

AppSheet permite criar aplicativos usando dados do Google Sheets, Excel e outras fontes, sem necessidade de codificação.

Zoho Creator

Zoho Creator é uma ferramenta para criar aplicativos personalizados para web e dispositivos móveis.

Thunkable ✕

Thunkable ✕ é a versão avançada do Thunkable com suporte para integrações mais complexas e personalizações.

GoodBarber

GoodBarber é uma plataforma para criar aplicativos móveis e PWAs (Progressive Web Apps) de forma visual e fácil.

AppGyver

AppGyver é uma plataforma para desenvolver aplicativos móveis e web apps com um editor visual poderoso.

Bildr

Bildr é uma ferramenta para criar aplicativos web interativos sem escrever código.

AppMakr

AppMakr permite criar aplicativos móveis para iOS e Android sem necessidade de habilidades de programação.

Swiftic

Swiftic é uma plataforma para criar aplicativos móveis com foco em pequenas empresas e empreendedores.

BiznessApps

BiznessApps é uma ferramenta para criação de aplicativos móveis personalizados para empresas.

Dropsource

Dropsource é uma plataforma para construir aplicativos móveis nativos sem escrever código.

Mendix

Mendix é uma plataforma de baixo código para desenvolvimento de aplicativos empresariais.

OutSystems

OutSystems é uma plataforma de desenvolvimento de aplicativos de baixo código para desenvolvimento rápido de aplicativos.

PowerApps

PowerApps é uma ferramenta da Microsoft para criar aplicativos de negócios personalizados sem escrever código.

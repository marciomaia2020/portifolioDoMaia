Apostila Selenium

1. Introdução ao Selenium

O Selenium é uma ferramenta poderosa para automação de navegadores web. Com ele, você pode automatizar tarefas repetitivas, realizar testes de ponta a ponta em aplicações web e até mesmo coletar dados de sites. Para mais informações, você pode acessar o site oficial do Selenium: selenium.dev.

2. Instalando o Selenium

Para instalar o Selenium, basta usar o pip:

pip install selenium

3. Configurando o WebDriver

O WebDriver é o componente do Selenium que interage diretamente com o navegador. Aqui está como configurar o WebDriver para o Chrome: 

from selenium import webdriver
from selenium.webdriver.common.by import By

# Caminho para o ChromeDriver
driver = webdriver.Chrome(executable_path='caminho/para/chromedriver')

# Navegar para uma URL
driver.get('https://www.google.com')

# Fechar o navegador
driver.quit()

Saída Esperada: O navegador Chrome deve abrir, navegar até o Google e então fechar.

4. Localizando Elementos

Você pode localizar elementos na página de várias maneiras:

# Localizando pelo ID
element = driver.find_element(By.ID, 'element_id')

# Localizando pelo Nome
element = driver.find_element(By.NAME, 'element_name')

# Localizando pelo XPath
element = driver.find_element(By.XPATH, '//*[@id="element_id"]')

# Localizando pelo CSS Selector
element = driver.find_element(By.CSS_SELECTOR, '.class_name')

Saída Esperada: Dependendo do método de localização, o Selenium irá interagir com o elemento correspondente na página.

5. Interagindo com Elementos

Aqui estão alguns exemplos de como interagir com os elementos localizados:

# Clicar em um botão
button = driver.find_element(By.ID, 'submit')
button.click()

# Escrever texto em um campo de entrada
input_field = driver.find_element(By.NAME, 'q')
input_field.send_keys('Selenium Python')

# Submeter um formulário
input_field.submit()

Saída Esperada: O Selenium deve clicar em botões, preencher campos de entrada e submeter formulários conforme programado.

6. Manipulando Clques e Interagindo com Tags HTML

O Selenium permite interagir com diversos elementos HTML, como links, botões, caixas de seleção e mais. Aqui estão algumas formas comuns de manipular cliques e interagir com diferentes tags:

Clicando em Links

# Clicar em um link pelo texto do link
link = driver.find_element(By.LINK_TEXT, 'Clique aqui')
link.click()

# Clicar em um link pelo link parcial
partial_link = driver.find_element(By.PARTIAL_LINK_TEXT, 'Clique')
partial_link.click()

Selecionando Caixas de Seleção e Botões de Rádio

# Selecionar uma caixa de seleção
checkbox = driver.find_element(By.ID, 'checkbox_id')
checkbox.click()

# Selecionar um botão de rádio
radio_button = driver.find_element(By.ID, 'radio_button_id')
radio_button.click()

Interagindo com Dropdowns

from selenium.webdriver.support.ui import Select

# Selecionar um item de um dropdown
dropdown = Select(driver.find_element(By.ID, 'dropdown_id'))
dropdown.select_by_visible_text('Opção 1')

# Selecionar por valor
dropdown.select_by_value('opcao1')

# Selecionar por índice
dropdown.select_by_index(0)

Saída Esperada: Dependendo da interação, o Selenium deve realizar as ações conforme especificado, como clicar em links, selecionar checkboxes e dropdowns.

7. Esperas Implícitas e Explícitas

As esperas são fundamentais para garantir que o Selenium aguarde o carregamento de elementos antes de interagir com eles:

Espera Implícita

driver.implicitly_wait(10)  # Espera implícita de 10 segundos

Espera Explícita

from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

# Espera explícita de até 10 segundos para que o elemento seja clicável
element = WebDriverWait(driver, 10).until(
    EC.element_to_be_clickable((By.ID, 'some_id'))
)
element.click()

Saída Esperada: O Selenium deve aguardar o tempo especificado até que o elemento esteja presente e interagir com ele.

8. Lidando com Janelas e Alertas

Alternando entre Janelas

# Captura o ID da janela atual
main_window = driver.current_window_handle

# Abre uma nova aba/janela
driver.execute_script("window.open('https://www.example.com');")

# Alterna para a nova aba/janela
driver.switch_to.window(driver.window_handles[1])

Lidando com Alertas

# Aceitar um alerta
alert = driver.switch_to.alert
alert.accept()

# Rejeitar um alerta
alert.dismiss()

Saída Esperada: O Selenium deve alternar entre janelas e lidar com alertas conforme necessário.

9. Capturando Screenshots

Você pode capturar screenshots da página atual:

# Capturar screenshot
driver.save_screenshot('screenshot.png')

Saída Esperada: Um arquivo de imagem chamado 'screenshot.png' deve ser criado no diretório especificado.

10. Fechando o Navegador

É importante fechar o navegador corretamente após os testes:

# Fecha a aba atual
driver.close()

# Fecha todas as abas e encerra o WebDriver
driver.quit()

Nota:

Cada navegador requer um WebDriver específico para funcionar com o Selenium:

  • ChromeDriver (para Google Chrome)
  • EdgeDriver (para Microsoft Edge)
  • GeckoDriver (para Mozilla Firefox)
  • SafariDriver (para Safari)

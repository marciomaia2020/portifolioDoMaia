Backup e Recuperação de Dados no Windows - Gabarito

Pergunta 1: O que é um backup?

  • A) Processo de mover arquivos para a lixeira
  • B) Cópia de segurança de arquivos importantes
  • C) Abertura de arquivos duplicados
  • D) Remoção de vírus do sistema

Resposta: B

Comentário: O backup é a cópia de segurança de arquivos e dados importantes para restaurá-los em caso de falha no sistema.

Pergunta 2: Qual ferramenta do Windows é usada para criar backups automáticos?

  • A) Microsoft Word
  • B) Paint
  • C) Histórico de Arquivos
  • D) Notepad

Resposta: C

Comentário: O Histórico de Arquivos no Windows permite o backup contínuo de arquivos automaticamente.

Pergunta 3: O que o ponto de restauração do sistema faz no Windows?

  • A) Remove arquivos não utilizados
  • B) Restaura o sistema a um estado anterior
  • C) Atualiza drivers de hardware
  • D) Instala novos softwares

Resposta: B

Comentário: Um ponto de restauração reverte as configurações do sistema a um estado anterior, útil para corrigir problemas.

Pergunta 4: Como o Backup do Windows é classificado em termos de tipo de backup?

  • A) Incremental
  • B) Completo
  • C) Diferencial
  • D) Todas as alternativas estão corretas

Resposta: D

Comentário: O Backup do Windows oferece várias opções, incluindo backups incrementais, diferenciais e completos.

Pergunta 5: O que é a ferramenta 'Restauro de Sistema' no Windows?

  • A) Uma ferramenta para desfragmentar o disco rígido
  • B) Um software para backup de fotos
  • C) Uma ferramenta que restaura o sistema para um ponto anterior
  • D) Um programa para edição de texto

Resposta: C

Comentário: O 'Restauro de Sistema' é usado para voltar as configurações do sistema a um ponto anterior sem alterar arquivos pessoais.

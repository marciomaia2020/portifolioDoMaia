Bibliotecas Python

  • Propósito: Flask é um microframework web para Python.
  • Usos:
    • Criação de aplicações web simples e complexas.
    • Desenvolvimento rápido de APIs.
    • Manutenção de aplicações leves com facilidade de extensão.
  • Características principais:
    • Simplicidade e flexibilidade.
    • Boa documentação e grande comunidade.
    • Suporte para extensões que podem adicionar funcionalidades, como autenticação de usuário e formulários.
  • Documentação do Flask
  • Propósito: Dash é um framework Python para construir aplicações web analíticas.
  • Usos:
    • Criação de dashboards interativos.
    • Visualização de dados de maneira dinâmica.
    • Integração fácil com bibliotecas de visualização de dados como Plotly.
  • Características principais:
    • Constrói aplicações web com a mesma facilidade de escrever scripts Python.
    • Suporte para callbacks interativos.
    • Facilita a criação de gráficos interativos complexos.
  • Documentação do Dash
  • Propósito: Pandas é uma biblioteca de manipulação e análise de dados.
  • Usos:
    • Manipulação de dados tabulares e séries temporais.
    • Limpeza e preparação de dados.
    • Análise estatística e operações de agregação.
  • Características principais:
    • DataFrames para manipulação de dados estruturados.
    • Ferramentas robustas para leitura e escrita de dados em diferentes formatos (CSV, Excel, SQL, etc.).
    • Operações de junção, fusão e agrupamento de dados.
  • Documentação do Pandas
  • Propósito: NumPy é uma biblioteca para computação numérica.
  • Usos:
    • Manipulação de arrays multidimensionais.
    • Operações matemáticas e estatísticas eficientes.
    • Base para muitas outras bibliotecas científicas em Python.
  • Características principais:
    • Arrays n-dimensionais (ndarray).
    • Funções matemáticas rápidas.
    • Suporte para álgebra linear, transformadas de Fourier e geração de números aleatórios.
  • Documentação do NumPy
  • Propósito: Matplotlib é uma biblioteca de plotagem para visualização de dados.
  • Usos:
    • Criação de gráficos estáticos, animados e interativos.
    • Visualização de dados científicos e de engenharia.
    • Integração com Jupyter notebooks.
  • Características principais:
    • Suporte para diversos tipos de gráficos (linha, barra, dispersão, etc.).
    • Personalização detalhada dos gráficos.
    • Ferramentas para salvar gráficos em diversos formatos.
  • Documentação do Matplotlib
  • Propósito: Scikit-learn é uma biblioteca para aprendizado de máquina.
  • Usos:
    • Implementação de algoritmos de aprendizado supervisionado e não supervisionado.
    • Preparação de dados e engenharia de features.
    • Avaliação de modelos de aprendizado de máquina.
  • Características principais:
    • Algoritmos prontos para uso, como regressão, classificação e clustering.
    • Ferramentas para validação cruzada e seleção de modelos.
    • Integração com outras bibliotecas de ciência de dados, como Pandas e NumPy.
  • Documentação do Scikit-learn
  • Propósito: TensorFlow e Keras são bibliotecas para aprendizado profundo (deep learning).
  • Usos:
    • Construção e treinamento de redes neurais.
    • Desenvolvimento de modelos de aprendizado profundo para tarefas como visão computacional e processamento de linguagem natural.
    • Deploy de modelos em diversas plataformas.
  • Características principais:
    • TensorFlow: Suporte extensivo para operações matemáticas e execução em GPUs.
    • Keras: Interface de alto nível para construção rápida de modelos de redes neurais.
    • Grande comunidade e abundância de recursos educacionais.
  • Documentação do TensorFlow
  • Documentação do Keras
  • Propósito: Requests é uma biblioteca para fazer requisições HTTP.
  • Usos:
    • Consumo de APIs REST.
    • Download de conteúdos web.
    • Envio de dados para servidores web.
  • Características principais:
    • Interface simples e intuitiva para enviar requisições HTTP.
    • Suporte para autenticação, sessões e cookies.
    • Tratamento de respostas HTTP.
  • Documentação do Requests
  • Propósito: BeautifulSoup é uma biblioteca para parsing de HTML e XML.
  • Usos:
    • Extração de dados de páginas web.
    • Parsing de documentos HTML e XML para análise.
    • Web scraping em combinação com bibliotecas como Requests e Selenium.
  • Características principais:
    • Facilidade de navegação e busca em árvores de documentos.
    • Suporte para diferentes parsers, como lxml e html.parser.
    • Tratamento robusto de documentos HTML malformados.
  • Documentação do BeautifulSoup
  • Propósito: pytest é uma biblioteca para testes em Python.
  • Usos:
    • Criação e execução de testes unitários.
    • Testes de integração e funcionais.
    • Automação de testes em pipelines de CI/CD.
  • Características principais:
    • Suporte para fixtures e plugins.
    • Mensagens de erro detalhadas e claras.
    • Flexibilidade e facilidade de uso.
  • Documentação do pytest
  • Propósito: Scrapy é um framework para web scraping.
  • Usos:
    • Extração de dados de sites de maneira estruturada.
    • Automação de navegação e coleta de informações web.
    • Criação de spiders para rastreamento web.
  • Características principais:
    • Alta performance e eficiência.
    • Suporte para pipelines de processamento de dados.
    • Fácil integração com outras bibliotecas e ferramentas.
  • Documentação do Scrapy

Pré-requisitos para usar o Node.js

  • Node.js (versão 18 ou superior)
  • npm ou yarn
  • Conta no Convex (gratuita)

Node.js

Baixe e instale o Node.js (versão 18 ou superior) diretamente do site oficial: https://nodejs.org/

Essa parte é o que você precisa já ter instalado/configurado no seu computador antes de rodar o sistema.

  • LTS (Long Term Support) → recomendado para a maioria dos projetos (mais estável).
  • Current → versão mais nova, mas pode mudar rápido (boa para testes ou novos recursos).

  1. Node.js (versão 18 ou superior)

    O Node.js é o ambiente que permite rodar JavaScript no servidor ou no seu PC.

    Ele é necessário porque o frontend desse projeto usa Vite (um bundler moderno) e o backend usa ferramentas em Node (como o Convex).

    👉 Você precisa instalar no mínimo a versão 18 (se já tiver uma mais nova, melhor ainda).

    Para conferir sua versão instalada:

    node -v

    Se aparecer algo como v18.x.x ou maior, está ok.

  2. npm ou yarn

    Tanto npm quanto yarn são gerenciadores de pacotes do Node.js.

    Eles servem para instalar e atualizar as dependências do projeto (as bibliotecas que o sistema precisa para funcionar).

    O npm já vem junto com o Node.js.

    O yarn é uma alternativa, mas precisa ser instalado separadamente: 

    npm install -g yarn

    No projeto, você pode usar um ou outro, mas geralmente se recomenda usar o npm por padrão.

👉 Em resumo:

  • Node.js → é o "motor" que faz o sistema rodar.
  • npm/yarn → são "os instaladores" que trazem as ferramentas do projeto.

🚀 Passo a Passo

  1. Clone/Download do Projeto 
    # Se você tem o código em um repositório
git clone [URL_DO_REPOSITORIO]
cd [NOME_DA_PASTA]

# Ou se você tem os arquivos localmente, navegue até a pasta
cd caminho/para/o/projeto
  2. Instalar Dependências 
    npm install
  3. Configurar o Convex
    1. Instalar Convex CLI globalmente 
      npm install -g convex
    2. Fazer login no Convex 
      npx convex login
    3. Inicializar o projeto Convex 
      npx convex dev
  4. Configurar Variáveis de Ambiente 
    CONVEX_DEPLOYMENT=seu-deployment-url
VITE_CONVEX_URL=https://seu-deployment.convex.cloud
  5. Deploy do Backend 
    npx convex deploy
  6. Inicializar Dados do Sistema 
    npx convex run users:initializeAdmin
  7. Iniciar o Frontend 
    npm run dev

🔧 Solução de Problemas

Erro de Autenticação

npx convex logout
npx convex login

Erro de Deploy

npx convex dev --once

Limpar Cache

rm -rf node_modules
npm install

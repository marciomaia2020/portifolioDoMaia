Manutenção de Computadores e Configuração do Setup BIOS

1. O que é BIOS?

A) Sistema Operacional
B) Programa que inicializa o hardware
C) Tipo de memória
D) Placa de rede

Resposta: B) Programa que inicializa o hardware

Comentário: A BIOS é responsável por verificar o hardware e iniciar o sistema operacional.

2. Qual a função da bateria CMOS?

A) Armazenar arquivos de usuário
B) Manter as configurações do sistema e da BIOS
C) Controlar a ventilação do sistema
D) Fornecer energia ao processador

Resposta: B) Manter as configurações do sistema e da BIOS

Comentário: A bateria CMOS mantém informações como data e hora mesmo quando o computador está desligado.

3. O que significa UEFI?

A) Universal Embedded Firmware Interface
B) Unified Extensible Firmware Interface
C) User Enterprise Framework Interface
D) Unified Electronics Firmware Installer

Resposta: B) Unified Extensible Firmware Interface

Comentário: UEFI é uma interface moderna que substitui a BIOS tradicional.

4. Como acessar a BIOS na maioria dos computadores?

A) Pressionando ALT+F4
B) Durante a inicialização, pressionando DEL ou F2
C) Ligando o PC com um CD de recuperação
D) Após o login no Windows

Resposta: B) Durante a inicialização, pressionando DEL ou F2

Comentário: Essas são as teclas mais comuns para acessar a BIOS durante o boot.

5. Qual a função da ordem de boot?

A) Controlar a temperatura dos discos
B) Definir a sequência de dispositivos para inicializar o sistema
C) Aumentar a performance do processador
D) Monitorar a energia da fonte

Resposta: B) Definir a sequência de dispositivos para inicializar o sistema

Comentário: A ordem de boot permite escolher de onde o sistema será iniciado, como HD ou pendrive.

6. Qual componente é mais propenso a falhas por superaquecimento?

A) Memória RAM
B) Disco rígido
C) Fonte de alimentação
D) CPU

Resposta: D) CPU

Comentário: O processador é sensível a altas temperaturas e precisa de bom resfriamento.

7. O que é manutenção preventiva?

A) Corrigir falhas quando ocorrem
B) Fazer backups diários
C) Realizar verificações e limpezas periódicas
D) Desinstalar programas desnecessários

Resposta: C) Realizar verificações e limpezas periódicas

Comentário: Manutenção preventiva evita problemas futuros e aumenta a vida útil do sistema.

8. O que é overclock?

A) Reduzir a velocidade do processador
B) Aumentar a frequência do processador além da especificação
C) Substituir a fonte de alimentação
D) Ativar o modo de segurança

Resposta: B) Aumentar a frequência do processador além da especificação

Comentário: Overclock pode aumentar o desempenho, mas aumenta o risco de superaquecimento.

9. Como verificar a integridade da memória RAM?

A) Usando MemTest86
B) Verificando na BIOS
C) Instalando um antivírus
D) Limpando os módulos de RAM

Resposta: A) Usando MemTest86

Comentário: MemTest86 é uma ferramenta especializada para testar a RAM.

10. O que é POST (Power-On Self Test)?

A) Um teste de desempenho do processador
B) A sequência de verificação do hardware ao iniciar o computador
C) Uma atualização de BIOS
D) Configuração de segurança da BIOS

Resposta: B) A sequência de verificação do hardware ao iniciar o computador

Comentário: O POST detecta falhas antes de o sistema carregar.

11. Qual alternativa de RAID oferece redundância e performance?

A) RAID 0
B) RAID 1
C) RAID 5
D) RAID 10

Resposta: D) RAID 10

Comentário: RAID 10 combina espelhamento e distribuição de dados, oferecendo segurança e velocidade.

12. O que significa AHCI na BIOS?

A) Automatic Hardware Configuration Interface
B) Advanced Host Controller Interface
C) Active Hard Drive Integration
D) Automated Hardware Cooling Interface

Resposta: B) Advanced Host Controller Interface

Comentário: AHCI melhora a performance de discos SATA e permite hot swapping.

13. Como restaurar as configurações padrão da BIOS?

A) Desinstalando a BIOS
B) Usando a opção 'Load Default Settings' na BIOS
C) Reiniciando o computador três vezes
D) Removendo o processador

Resposta: B) Usando a opção 'Load Default Settings' na BIOS

Comentário: Restaurar as configurações pode corrigir problemas de inicialização.

14. O que deve ser verificado em caso de não inicialização do sistema?

A) A fonte de alimentação
B) A temperatura do CPU
C) O número de jogos gerados
D) A cor da placa-mãe

Resposta: A) A fonte de alimentação

Comentário: Problemas de energia são comuns em falhas de inicialização.

15. Qual é a importância de atualizar a BIOS?

A) Aumentar a capacidade do disco rígido
B) Melhorar a compatibilidade e a estabilidade do sistema
C) Reduzir a velocidade do processador
D) Alterar a configuração da memória RAM

Resposta: B) Melhorar a compatibilidade e a estabilidade do sistema

Comentário: Atualizações podem corrigir bugs e adicionar suporte a novos hardware.

O que é um Projeto de LANs e WANs?

Projeto de LANs e WANs - Refere-se ao

Quando se fala de "Projeto de LANs e WANs (especificação de infraestrutura, cabeamentos, ativos, aspectos de gerenciamento, intranet, extranets e internet)", é necessário estudar os seguintes tópicos:

  • Infraestrutura de redes: Este tópico inclui o estudo dos componentes físicos de uma rede, como cabos, switches, roteadores, servidores e pontos de acesso sem fio. É importante entender as diferentes tecnologias de cabeamento e como elas afetam o desempenho da rede. Também é importante entender como os diferentes dispositivos de rede se comunicam entre si.
  • Cabeamento: Este tópico inclui o estudo dos diferentes tipos de cabos de rede e como eles são instalados. É importante entender as diferentes características dos cabos, como a velocidade de transmissão, o alcance e a resistência a interferências. Também é importante entender as diferentes técnicas de cabeamento, como o cabeamento estruturado.
  • Ativos: Este tópico inclui o estudo dos diferentes tipos de ativos de rede, como servidores, estações de trabalho, impressoras e dispositivos de armazenamento. É importante entender as diferentes características dos ativos, como o desempenho, o consumo de energia e os requisitos de segurança. Também é importante entender como os ativos são gerenciados.
  • Aspectos de gerenciamento: Este tópico inclui o estudo das diferentes ferramentas e técnicas utilizadas para gerenciar uma rede. É importante entender como as redes são monitoradas e como os problemas são resolvidos. Também é importante entender como as redes são planejadas e implantadas.
  • Intranets, extranets e internet: Este tópico inclui o estudo das diferentes redes que podem ser implementadas em uma organização. É importante entender como as intranets, extranets e internet são usadas para compartilhar informações e recursos.

Além desses tópicos, é importante também estudar os seguintes conceitos:

  • Topologias de rede: As diferentes maneiras como os dispositivos de rede são conectados entre si.
  • Protocolos de rede: As regras que definem como os dispositivos de rede se comunicam entre si.
  • Segurança de rede: As medidas tomadas para proteger a rede contra acessos não autorizados.

Ao estudar esses tópicos, você estará preparado para projetar redes LAN e WAN eficazes que atendam às necessidades de sua organização.

    • "Network Design Essentials" de Keith Barker

    • "Designing and Implementing Enterprise Networks" de Kevin Miller
  • Cursos online: Existem muitos cursos online disponíveis sobre projeto de redes LAN e WAN. Alguns exemplos incluem:
    • "Network Design" da Udemy
    • "Cisco CCNA: Network Design" da Pluralsight
    • "Designing Enterprise Networks" da CBT Nuggets
  • Certificações: Existem várias certificações disponíveis que podem ajudá-lo a demonstrar suas habilidades em projeto de redes LAN e WAN. Algumas exemplos incluem:
    • CCNA: Certified Cisco Network Associate
    • CCNP: Certified Cisco Network Professional
    • CCIE: Certified Cisco Internetwork Expert

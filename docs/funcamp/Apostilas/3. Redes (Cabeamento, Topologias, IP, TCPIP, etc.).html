Redes (Cabeamento, Topologias, IP, TCPIP, etc.)

1. O que é uma topologia de rede?

Topologia de rede é a forma como os dispositivos estão fisicamente ou logicamente conectados, como estrela, anel, barramento e malha.

2. O que é uma rede LAN?

LAN (Local Area Network) é uma rede local restrita a uma pequena área, como uma casa ou escritório.

3. O que é uma rede WAN?

WAN (Wide Area Network) é uma rede de longa distância que conecta várias redes locais, como a Internet.

4. Qual é a função de um switch?

O switch conecta dispositivos em uma rede local e direciona pacotes de dados para o destino correto.

5. O que é um roteador?

O roteador conecta diferentes redes e define a melhor rota para o tráfego de dados.

6. O que é IP?

IP (Internet Protocol) é o protocolo que define como os pacotes de dados são endereçados e roteados na Internet.

7. Qual a diferença entre IPv4 e IPv6?

IPv4 usa endereços de 32 bits, enquanto IPv6 utiliza endereços de 128 bits, permitindo mais dispositivos conectados.

8. O que é máscara de sub-rede?

Máscara de sub-rede define quais bits do endereço IP representam a rede e quais identificam os dispositivos.

9. O que é gateway padrão?

É o dispositivo que encaminha o tráfego de uma rede local para outras redes, geralmente um roteador.

10. Qual é a função do protocolo DHCP?

DHCP (Dynamic Host Configuration Protocol) atribui automaticamente endereços IP aos dispositivos na rede.

11. O que é DNS?

DNS (Domain Name System) traduz nomes de domínio em endereços IP.

12. O que é TCP/IP?

TCP/IP é um conjunto de protocolos usados na comunicação entre dispositivos na Internet.

13. Qual é a diferença entre TCP e UDP?

TCP é orientado a conexão e garante entrega, enquanto UDP é mais rápido, mas não garante a entrega.

14. O que é MAC Address?

É o endereço físico único de uma interface de rede, atribuído ao dispositivo pelo fabricante.

15. O que é um firewall?

Firewall é um sistema de segurança que monitora e controla o tráfego de rede com base em regras predefinidas.

16. Como funciona o protocolo FTP?

FTP (File Transfer Protocol) é usado para transferir arquivos entre dispositivos em uma rede.

17. O que é ping?

Ping é uma ferramenta que envia pacotes ICMP para verificar a conectividade entre dispositivos.

18. O que é NAT?

NAT (Network Address Translation) permite que vários dispositivos usem um único endereço IP público.

19. O que é VLAN?

VLAN (Virtual Local Area Network) é uma rede lógica criada dentro de uma rede física.

20. Qual é a função do protocolo ARP?

ARP (Address Resolution Protocol) traduz endereços IP em endereços MAC.

21. O que é uma rede peer-to-peer?

É uma rede onde todos os dispositivos têm responsabilidades iguais e podem compartilhar recursos entre si.

22. O que é um servidor proxy?

Proxy é um servidor intermediário entre o cliente e a Internet, usado para controle e segurança.

23. O que é uma VPN?

VPN (Virtual Private Network) cria uma conexão segura entre redes através da Internet.

24. Como funciona o protocolo HTTPS?

HTTPS (HyperText Transfer Protocol Secure) criptografa a comunicação entre cliente e servidor para segurança.

25. O que é um endereço IP público?

É um endereço IP acessível pela Internet, atribuído a dispositivos que precisam de acesso externo.

26. O que é um endereço IP privado?

É um endereço IP usado dentro de uma rede local, não acessível diretamente pela Internet.

27. O que é o modelo OSI?

O modelo OSI define sete camadas para padronizar a comunicação entre dispositivos de rede.

28. O que é a camada de enlace do modelo OSI?

Essa camada é responsável pela comunicação entre dispositivos na mesma rede local.

29. O que é um ataque DDoS?

DDoS (Distributed Denial of Service) é um ataque que sobrecarrega um servidor com tráfego para tirá-lo do ar.

30. Como se mede a velocidade de uma rede?

A velocidade de uma rede é medida em bits por segundo (bps), como Mbps ou Gbps.

Servidores Locais


WampServer

O WampServer é uma plataforma de desenvolvimento web no Windows, que permite criar aplicações web com Apache, PHP e MySQL.

XAMPP

O XAMPP é uma distribuição Apache fácil de instalar contendo MySQL, PHP e Perl.

MAMP

O MAMP é uma solução gratuita que pode ser instalada em macOS e Windows com Apache, MySQL e PHP.

Laragon

O Laragon é uma plataforma de desenvolvimento poderosa e portátil para Windows, que inclui Apache, MySQL, PHP e ferramentas adicionais.

AMPPS

O AMPPS é um pacote de servidor local que inclui Apache, MySQL, MongoDB, PHP, Perl, e Python. Facilita a instalação de várias aplicações web como WordPress, Joomla, e Drupal.

EasyPHP

O EasyPHP é uma pilha de servidor para Windows que oferece um ambiente de desenvolvimento completo, incluindo Apache, MySQL, e PHP.

Local by Flywheel

O Local by Flywheel é um ambiente de desenvolvimento local para WordPress, oferecendo uma interface amigável e diversas ferramentas específicas para desenvolvedores WordPress.

Devilbox

O Devilbox é um ambiente de desenvolvimento flexível e extensível baseado em Docker, que suporta múltiplas versões de PHP, MySQL, e outras tecnologias.

Docker

O Docker não é um servidor em si, mas uma plataforma que permite a criação e gestão de contêineres, permitindo a configuração de ambientes de desenvolvimento personalizados com várias tecnologias (incluindo servidores web).

Uniform Server

O Uniform Server é uma solução de servidor WAMP portátil para Windows que inclui Apache, MySQL, PHP, e Perl.

Bitnami

O Bitnami oferece uma ampla variedade de pacotes de software empacotados para servidores, incluindo stacks LAMP, WAMP, MAMP, além de pilhas específicas para aplicações como WordPress, Joomla, Drupal, etc.

UwAmp

O UwAmp é um servidor WAMP que inclui Apache, MySQL, e PHP, com uma interface amigável para administração.

OpenServer

O OpenServer é uma plataforma de desenvolvimento local que inclui Apache, MySQL, PHP, Nginx, e outros servidores, além de suporte a diferentes linguagens de programação.

Vagrant

O Vagrant utiliza arquivos de configuração para criar ambientes de desenvolvimento portáteis e replicáveis, frequentemente usados em conjunto com VirtualBox ou outros provedores de virtualização.

VestaCP

O VestaCP é um painel de controle de hospedagem gratuito e de código aberto que inclui Nginx, Apache, MySQL, e suporte a PHP.

CyberPanel

O CyberPanel é um painel de controle baseado em OpenLiteSpeed que oferece um ambiente de desenvolvimento completo com suporte para Apache, MySQL, e PHP.

Caddy

O Caddy é um servidor web moderno e fácil de configurar, com suporte embutido para HTTPS. Pode ser usado em conjunto com outras ferramentas para criar um ambiente de desenvolvimento completo.

Comandos Básicos de Linux e DOS

Pergunta 1: O que faz o comando ls no Linux?

Exibe uma lista dos arquivos e diretórios no diretório atual, equivalente ao dir no DOS.

Pergunta 2: Como se navega entre diretórios no Linux e no DOS?

Linux: cd <diretório>
DOS: cd <diretório>

Ambos os comandos mudam para o diretório especificado.

Pergunta 3: Qual o comando para exibir o caminho completo do diretório atual?

Linux: pwd
DOS: cd

No DOS, o comando cd sem argumentos exibe o diretório atual.

Pergunta 4: Como listar arquivos ocultos no Linux?

ls -a

No Linux, arquivos que começam com ponto (.) são ocultos.

Pergunta 5: Como copiar arquivos em Linux e DOS?

Linux: cp origem destino
DOS: copy origem destino

Pergunta 6: Como mover ou renomear arquivos?

Linux: mv origem destino
DOS: move origem destino

Renomear e mover usam o mesmo comando em ambos.

Pergunta 7: Como remover arquivos e diretórios?

Linux: rm arquivo e rm -r diretório
DOS: del arquivo e rmdir /s diretório

Pergunta 8: Qual o comando para criar diretórios?

Linux: mkdir <nome>
DOS: mkdir <nome> ou md <nome>

Pergunta 9: Como visualizar o conteúdo de um arquivo no terminal?

Linux: cat arquivo
DOS: type arquivo

Pergunta 10: Como buscar texto dentro de arquivos?

Linux: grep 'texto' arquivo
DOS: find "texto" arquivo

Pergunta 11: Como visualizar processos em execução?

Linux: ps ou top
DOS: tasklist

Pergunta 12: Como finalizar um processo?

Linux: kill <PID>
DOS: taskkill /PID <PID> /F

Pergunta 13: Como verificar o espaço em disco?

Linux: df -h
DOS: dir

Pergunta 14: Como monitorar o uso de memória?

Linux: free -h
DOS: tasklist

Pergunta 15: Como mudar permissões de arquivos no Linux?

chmod 755 arquivo

Pergunta 16: Como verificar as permissões de um arquivo no Linux?

ls -l

Pergunta 17: Como criar um arquivo vazio?

Linux: touch nome_do_arquivo
DOS: copy nul nome_do_arquivo

Pergunta 18: Como reiniciar a máquina via terminal?

Linux: reboot
DOS: shutdown /r /t 0

Pergunta 19: Como desligar a máquina via terminal?

Linux: shutdown -h now
DOS: shutdown /s /t 0

Pergunta 20: Como ver informações sobre a rede?

Linux: ifconfig ou ip a
DOS: ipconfig

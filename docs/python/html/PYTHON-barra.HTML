New-Item -ItemType File -Path $PROFILE -Force

Local do arquivo de perfil

C:\Users\\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1

Editar o arquivo de perfil

notepad $PROFILE

Chamar o arquivo de perfil

. $PROFILE

Extensões de Arquivo e Programas Compatíveis

.docxDocumento de texto do Microsoft Word, com formatação avançada, imagens e tabelas.Microsoft Word, Google Docs, LibreOffice
.pdfDocumento de formato portátil, visualizável em qualquer dispositivo com layout preservado.Adobe Acrobat Reader, Foxit Reader, Chrome
.mdArquivo Markdown, usado para formatação leve de textos.VS Code, Typora, Atom
.rtfRich Text Format, suporta formatação de texto e imagens.Microsoft Word, WordPad, LibreOffice
.wpsArquivo de documento do WPS Office, semelhante ao Word.WPS Office, Microsoft Word, LibreOffice

.svgImagem vetorial escalável, usada para gráficos e ícones sem perder qualidade.Inkscape, Adobe Illustrator, VS Code
.gifImagem animada em loop, comumente usada em mídias sociais e sites.GIMP, Photoshop, Web Browsers
.bmpImagem de mapa de bits, sem compressão e com qualidade máxima.Paint, Photoshop, GIMP
.tiffImagem em formato TIFF, usado para imagens de alta qualidade, como fotografias e impressão.Adobe Photoshop, GIMP, Windows Photos

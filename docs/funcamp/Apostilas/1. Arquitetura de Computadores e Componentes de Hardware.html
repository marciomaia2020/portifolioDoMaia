Arquitetura de Computadores - Quiz

1. O que é a Unidade Central de Processamento (CPU)?

A) Uma unidade de armazenamento secundário

B) A parte do computador que executa instruções

C) Um periférico de entrada e saída

D) Um sistema de refrigeração

Resposta: B

Comentário: A CPU executa as instruções de programas, controlando o funcionamento de todo o computador.

2. Qual é a função principal da memória RAM?

A) Armazenar dados permanentemente

B) Fornecer armazenamento rápido e temporário para o processador

C) Proteger o sistema contra falhas elétricas

D) Controlar dispositivos de entrada e saída

Resposta: B

Comentário: A RAM é uma memória volátil, usada para armazenar dados e programas temporariamente enquanto o computador está ligado.

3. O que é o ciclo de busca e execução (fetch-execute cycle)?

A) O processo de desligamento de um computador

B) O processo de executar uma operação aritmética

C) O ciclo de busca, decodificação e execução de instruções na CPU

D) O ciclo de leitura e gravação de dados no disco

Resposta: C

Comentário: O ciclo fetch-execute descreve como a CPU busca uma instrução da memória, decodifica e executa.
